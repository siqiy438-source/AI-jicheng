## 🐛 常见问题排查

### 输入框仍然放大
- 检查 viewport meta 标签
- 检查输入框字体大小 ≥ 16px
- 清除浏览器缓存

### PWA 无法安装
- 确保使用 HTTPS（或 localhost）
- 检查 manifest.json 配置
- 检查 Service Worker 注册
- 检查图标文件存在

### 性能问题
- 检查图片是否压缩
- 检查是否启用懒加载
- 检查动画是否过多
- 使用 Performance 面板分析

### 布局问题
- 检查安全区域变量
- 检查响应式断点
- 检查 flexbox/grid 布局
- 使用 DevTools 检查元素

## 📊 性能监控

### 使用 Lighthouse CI
```bash
# 安装 Lighthouse CI
npm install -g @lhci/cli

# 运行审计
lhci autorun --collect.url=http://localhost:5173
```

### 使用 WebPageTest
1. 访问 https://www.webpagetest.org/
2. 输入网站 URL
3. 选择移动设备
4. 选择 3G 网络
5. 运行测试
